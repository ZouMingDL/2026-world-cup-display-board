**b) 数据获取 useEffect**

```ts
useEffect(() => {
  if (activeTab === 'schedule' && scheduleMatches.length === 0) {
    setScheduleLoading(true);
    fetchAllMatches()
      .then(matches => {
        setScheduleMatches(matches);
        setScheduleLoading(false);
      })
      .catch(err => {
        setScheduleError('赛程数据加载失败，请稍后重试');
        setScheduleLoading(false);
      });
  }
}, [activeTab, scheduleMatches.length]);
```

**c) 赛程数据按日期分组 useMemo**

```ts
const groupedSchedule = useMemo(() => {
  const filtered = scheduleMatches.filter(m => {
    if (scheduleFilter === 'upcoming') return m.status === 'NS';
    if (scheduleFilter === 'finished') return m.status === 'FT';
    return true;
  });
  const groups: Record<string, WorldCupMatch[]> = {};
  for (const match of filtered) {
    const dateKey = match.date;
    if (!groups[dateKey]) groups[dateKey] = [];
    groups[dateKey].push(match);
  }
  return Object.entries(groups).sort(([a], [b]) => a.localeCompare(b));
}, [scheduleMatches, scheduleFilter]);
```

```ts
function formatDate(dateStr: string) {
  const d = new Date(dateStr);
  return `${d.getMonth() + 1}月${d.getDate()}日 ${'日一二三四五六'[d.getDay()]}`;
}
```

## UI 设计细节

### 赛程卡片样式

```
┌─────────────────────────────┐
│  🇲🇽 墨西哥    VS    南非 🇿🇦  │
│      1 : 0                 │
│  ⏰ 2026-06-11 19:00       │
│  📍 Estadio Azteca         │
│  状态: 已结束               │
└─────────────────────────────┘
```

- 卡片背景白色，圆角，轻微阴影
- 比赛状态标签：未开始=灰色，已结束=绿色，进行中=橙色（动态闪烁）
- 队徽使用 TheSportsDB 返回的 `strHomeTeamBadge` / `strAwayTeamBadge`

### 日期分组头部

- 粘性定位（sticky），滚动时固定在顶部
- 显示格式：「6月11日 周三 · 10场比赛」

3. 确认赛程数据按日期分组显示
4. 切换筛选按钮（全部/未开始/已结束）确认过滤正常
5. 确认队徽图片正常加载
6. `npm run lint`（即 `tsc --noEmit`）确认无类型错误
